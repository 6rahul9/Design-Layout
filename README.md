## Terminal
| Code                | Description                                              | 
| ------------------- | -------------------------------------------------------- |
| `npm install`       | Install dev dependencies                                     |
| `npm run start`       | Runs app with development mode on `http://localhost:1234`|
| `npm run build`     | Builds the app for production to the `dist` folder.      |
